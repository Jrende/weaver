How to use this thing

Weave settings

Here you can set the width and height of the weave and the amount of treadles and shafts.

The inputs support evaluating expressions, so for example typing 4*2 turns is to 8.

Yarns

Add or remove yarns from the weave. These are used as the palette when setting the colors of the warp of weft threads

Threading and treadle order

Fill out the threading or treadle order

The grid can be used to create a smaller pattern which can be used to fill the draft.

Mirrored repetition

If mirrored repetition is checked the pattern will be mirrored every other repeat
